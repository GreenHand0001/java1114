# 作业管理系统 java1114

## 项目概述

作业管理系统是一款基于Java + SpringBoot + Vue的前后端分离系统。它实现了对作业发布、提交、批改以及成绩管理的全面流程控制，并采用MVC分层设计，确保系统的高效性与可维护性。

## 技术栈

- **后端**：Java + SpringBoot + MyBatis
- **前端**：Vue.js + Element-UI
- **数据库**：MySQL
- **开发环境**：IDEA、JDK 1.8、Maven

## 功能模块

### 系统角色
- 管理员
- 教师
- 学生

各角色权限隔离，确保数据安全。

### 核心功能
- **主页**：系统概览与快捷入口
- **作业流程**
  - 教师：发布作业、批改作业
  - 学生：提交作业、查看成绩
- **系统管理**
  - 用户管理
  - 角色管理
  - 菜单管理
  - 文件管理

### 特色功能
- **前后端分离**：基于API接口交互，实现业务逻辑与界面的解耦
- **响应式设计**：自动适配PC及移动端浏览器
- **数据安全**：基于角色的权限控制（RBAC）

## 运行环境

- **操作系统**：Windows / Mac（跨平台支持）
- **依赖环境**：JDK 1.8、MySQL、Maven、Node.js

## 部署步骤

1. **环境准备**：安装JDK 1.8、MySQL、Maven、Node.js，配置IDEA开发环境
2. **数据库初始化**：执行SQL脚本创建表结构
3. **后端部署**：
   - 导入Maven项目，编译打包（`mvn clean install`）
   - 启动SpringBoot应用（`java -jar`）
4. **前端部署**：
   - 安装Vue依赖（`npm install`）
   - 编译打包（`npm run build`）
   - 部署到Nginx或直接运行开发服务器
5. **访问系统**：在浏览器中打开`http://localhost:端口号`
